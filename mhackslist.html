PSP2 - 60FPS Patch

This Phantasy Star Portable 2 60FPS hack allows the game to run at 60 fps natively. This patch works in both single player and multiplayer and vastly improves the game. This cheat is easiest used with an emulator such as PPSSPP, simply use the below code in the emulator's 'cheats' function. You can also patch a PSP ISO however the PSP itself struggles to maintain anything above 30fps.

CW Cheat / Patch
_S ULUS-10529
_G Phantasy Star Portable 2
_C1 60fps
_L 0x00630B6C 0x00000001
_L 0x20662FF4 0x42700000
_L 0x20662FF8 0x0000003C
_L 0x20662FFC 0x3F800000
_L 0x20663000 0x00000001
_L 0x20663004 0x3F000000
_L 0x20663008 0x3C888889
_C0 30fps
_L 0x00630B6C 0x00000002
_L 0x20662FF4 0x41F00000
_L 0x20662FF8 0x0000001E
_L 0x20662FFC 0x40000000
_L 0x20663000 0x00000002
_L 0x20663004 0x3F800000
