Succubus Heart (Skyrim Mod)

Succubus Heart is an adult gameplay mod focused on creating new ways to approach playing Skyrim with both new progression and new game mechanics. These mechanics give the player the power of a succubus enabling the use of life-drain and great sacrificial powers. Succubus Heart is built around the idea of being a complimentary mod and should be compatible with nearly every mod out there!

How To Install (+Requirements)

What's Included
- Docs (Read Me - ChangeLog - Source Notes)
- Data (Everything for the core mod)
- Optional Uncapper Settings (Customized Skyrim Com. Unc. Settings)

Requrements
- SKSE 1.07.03 or newer http://skse.silverlock.org
- SexLab v1.62 or newer
- Sky UI 5.1 or newer http://www.nexusmods.com/skyrim/mods/3863/?

Installing : Simply copy Succubus Heart's "Data" folder to your Skyrim install, merging with that "Data" folder, then install any optional content you wish to use.

Updating : If you're using version 2.73x or below you'll need to do a 'clean' save before moving to 3.0+.

Uninstalling : Disable the mod then perform a clean save. After the clean save delete the "Succubus Heart.esp" and all files pre-fixed with "SuH_" and other files from the Succubus Heart mod from your Skyrim's "Data" folder. It is recommended to either load a game from before you had Succubus Heart installed or start a new game after uninstalling.

Lucrative Augments (Fallout 4)

NOTICE : Currently in early stages of development and is considered feature incomplete.

Lucrative Augments adds a new (self-contained) Augments System to Fallout 4! These Augments are crafted at the chem station and, when successfully applied to the player, add various passives that can either help or harm you! This allows some additional character alterations and a bit of its own depth! Think of it as an additional perk system that has it's own limits and scaling.

How To Install (+Requirements)

What's Included
- Docs (Read Me - ChangeLog - Source Notes)
- Data (Everything for the core mod)

Requrements
- Fallout 4 Base Game (Also supports DLCs)
- F4SE - https://f4se.silverlock.org/
- (optional) MCM to configure Modules and check status - (Nexus)
- (optional) AAF for 'Adultify' integration - (Nexus)

Installing : Copy all files from the "Data" folder included with Lucrative Augments into your Fallout 4 install's Data folder. Enable the mod in your mod-manager of choice.

Uninstalling : Disable the mod in your mod-manager of choice. After this you can delete all the files from Lucrative Augments from your Fallout 4 install. NOTE : Any direct stat/visual modifications to characters will stay even after the mod is uninstalled.
