What is Project Panic?
Project Panic is a 2D platformer where the goal is to collect ideas and memories so that when the character wakes up he/she can finish his/her graduation project!
History of Project Panic
Back in High School I was required to do a 'graduation project' in order to graduate. I knew I wanted to pursue game development so I went ahead and built a small game which happened to be Project Panic. The game was developed in a 1 week timeframe using the XNA framework, a lot of assets were gathered under the assumption of 'Education' copyright rules. I'd go back and change a bunch of things however the source code and all its assets have been long lost.

Troubleshooting / F.A.Q.
The game won't download, what do I do?
Make sure you're using a modern and up-to-date browser such as Chrome or Firefox.

The game won't launch and shows an error?
The most common issue with launching the game is an out-of-date or incomplete DirectX install. Either run the DirectX Web Installer found here or install an up-to-date DirectX 9 standalone. I recommend using the web installer.
